# Sorting Algorithms

## Languages

### Java
Java sorting algorithms are implemented to sort arrays, lists and array lists.
Following sorts implemented:
- Bogo sort
- Bubble sort
- Counting sort
- Heap sort
- Instert sort
- Mergesort
- Quick sort
- Selection sort

### Python
Python sorting algorithms are implemented to sort lists.
Following sorts implemented:
- Bogo sort
- Bubble sort
- Counting sort
- Heap sort
- Instert sort
- Mergesort
- Quick sort
- Selection sort
